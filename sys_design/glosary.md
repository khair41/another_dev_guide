## ACID
- Atomicity
    - Each transaction is fully completed or not executed at all
- Consistency
    - Each transaction brings the DB from one valid state to another
- Isolation
    - Concurrent transactions do not interfere with each other
- Durability
    - Commited data cant be undone even if the system fails

## Request throttling
Request was limited or temporarily rejected because you exceeded the allowed number of requests within a specific time frame. 
This is a control mechanism used by services to prevent a single user or application from overwhelming a system, ensuring fair access for all users and maintaining performance and security
