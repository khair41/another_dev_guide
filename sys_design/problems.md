# Problema: Diseña un sistema tipo "Realtime Collaborative Notes"

## Contexto:

Quieren construir un servicio web y móvil donde múltiples usuarios puedan editar simultáneamente la misma nota en tiempo real como Google Docs. Los usuarios pueden:

- Crear, leer, actualizar y borrar notas.

- Invitar a otros usuarios a colaborar en sus notas.

- Ver cambios de otros usuarios en tiempo real.

- Tener historial de cambios (versionado).


Se requiere alta disponibilidad, baja latencia en la actualización de cambios en tiempo real, escalable para millones de usuarios y notas, notificaciones de cambios.
